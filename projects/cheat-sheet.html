Ruby BASICS Cheat Sheet
Concept Syntax Description
Comment # I am a comment! Ruby ignores everything that is marked as a comment.
Variables variablename = some_value A variable can hold almost any type of value including numbers, strings, arrays, and hashes.
p p 2 + 2 Returns the evaluated code and prints to the console.
puts puts "Text to output" Returns nil and prints the output to the console with a new line.
print print "Printed string here!" Returns nil and prints the output to the console.
Call a method object.method(arguments) We are sending the object some kind of message/command and are waiting for its response. It may have no arguments or multiple arguments, depending on the message.
Define a method def name(parameter) puts "Hey" + parameter end Methods are basically reuseable units of behavior, small and focused on implementing that specific behaviour.
Equality object == anotherobject With two equal signs you can check if two things are the same. If so, true will be returned; otherwise, the result will be false.
Inequality object != anotherobject Inequality is the inverse to equality, it results in true when two values are not the same and it results in false when they are the same.
Integer 5, 26, 75 A whole number, or a number that does not contain any decimal points.
Float 5.6, 72.33, 100.1 Short for floating-point numbers, floats are numbers that contain decimal points (are not whole numbers).
Boolean true & false In Ruby, there are two boolean values: true and false. You use logical operators (below) to compare to boolean values.
String "I am a string of text!" Used to hold textual information. May contain single characters, words, sentences or a whole book. However you may, just think of them as an ordered collection of characters.
Add/Subtract/Multiply/Divide + - * / These are the arithmetic operators available in Ruby, for more information on these see the link at the bottom of the page to the ruby docs.
exponentiation 5**2 = 25 This is how you square (or get the power of) a number in Ruby.
modulus 7 % 3 = 1 Gives you the remainder after division by a number. So for the example shown, 7 / 3 = 2 with a remainder of 1.
Other Comparison Operators > >= < <= Greater than, greater than or equal to, less than, & less than or equal to are additional comparison operators (in addition to equality and inequality listed above. Ruby compares strings in lexicographical ordering.
and boolean1 && boolean2 Another logical operator used to compare boolean values in Ruby.
or boolean1 || boolean2 Another logical operator used to compare boolean values in Ruby.

For more on Ruby, check out the Ruby Docs here!
