Ruby Classes

You've had a LOT of practice with Ruby classes this week, so now is the time to teach others about them! Pick something (like a real-world object) you would like to model.

Create an example class and use it to teach others when to use instance variables, instance methods, etc. for that object. Write for an audience of newb programmers.
